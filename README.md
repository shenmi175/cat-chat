## 介绍📝

**没头脑的马屁精** 是一款运行在桌面上的互动型虚拟电子宠物程序。它是一个可爱的小伙伴，每天最大的乐趣就是观察主人的电脑屏幕、记录主人的喜好，并运用基于大模型的精湛“马屁技巧”为主人提供源源不断的情绪价值！

作为 v1.0 **Demo 测试版本**，本项目已完成了核心的 Electron 透明窗口渲染、DeepSeek 云端模型接入、硬件状态感知以及基础的记忆追踪功能，用于验证桌面电子宠物的可行性与核心趣味体验。

## 🚀 下一版本演进路线 (Roadmap)

本 Demo 验证成功后，开发团队将在下一个大版本中重点推进以下特性的演进与重构：

### 1. 🎨 视觉与生命力重构 (Live2D / 骨骼动画)
- **告别颜文字**：移除现有的 `(=^･ω･^=)` 纯文本展示，全面接入 Live2D 或 Spine 骨骼动画引擎。
- **动态交互**：实现呼吸、眨眼、睡觉等待机动画。当鼠标指针在它头上划过时，触发“抚摸”动画效果及好感度系统。
- **换装系统**：支持为它穿戴不同的帽子、项圈或眼镜。

### 2. 🗣️ 感官与语音升级 (TTS)
- **真声合成**：接入 Edge-TTS 或类似语音模型，让它夸赞不再只停留在屏幕上，而是用亲切的声音直击你的灵魂。
- **环境音场**：增加专属的卖萌音效、系统提示声、以及弹出控制台时的 UI 科技音效。

### 3. 🧠 大脑与感知边界拓展
- **深度系统集成**：除了当前窗口和电量，未来将尝试读取剪贴板、浏览器标签页焦点、本地天气预报以及系统日历日程（如：“主人！今天主人有一个重要会议，一定能大展宏图！”）。
- **长期记忆向量化**：将当前简单的 JSON 数组记忆库重构为本地轻量级向量数据库（如 ChromaDB/SQLite-VSS），支持基于上下文语义的海量记忆动态检索。

### 4. ⚙️ 工程化与跨平台优化
- **沙盒与静默更新**：实现 Electron 客户端的自动静默更新机制。
- **平台兼容**：修复 Linux Wayland 环境下获取焦点的原生限制，优化 Windows 11 下的防边缘溢出逻辑，甚至适配 MacOS 顶栏常驻特性。

## 核心亮点 ✨

- 🐾 **全天候悬浮相伴**：基于 Electron 的无边框透明窗口，无论你在打游戏还是写代码，它都在桌面的一角默默看着你。
- 🧠 **深度模型赋能**：完美接入 **DeepSeek API**（支持 Chat 与 Reasoner 双模型切换），对答如流，绝不呆板。
- 👁️ **系统级电脑感知**：它能“看到”你现在正在用的软件、窗体标题，甚至知道你的电脑还剩多少电量。它会在你电量见底或者沉迷某个游戏时，突然跳出来对你说几句话。
- 💾 **自动潜意识记忆**：不仅仅是一个每次都重启的对话机器人。它的大脑自带分析系统，在与你聊天的过程中，会自动从你的话语中提取出关键偏好（“主人爱喝酸奶”、“主人喜欢叫 Lucas”），并永久保存在记忆库中，下一秒立刻拿来拍马屁！
- 🎛️ **1200x800 极客控制台**：右键它即可唤出深色玻璃态（Glassmorphism）的统一控制面板，自由穿梭于 **通用设置、记忆管理、回溯历史档案** 与 **系统实时监测室** 之间。

## 快速开始 🚀

### 1. 环境准备
请确保你的电脑上已经安装了 [Node.js](https://nodejs.org/) (推荐 v18+)。

### 2. 克隆与安装依赖
```bash
# 获取源码 (如果已在这个目录下则跳过)
git clone https://github.com/your-username/pet-chat.git
cd pet-chat

# 安装所有依赖 (包含 React, Electron, Vite 及系统探测包)
npm install
```

### 3. 本地运行调试
```bash
# 启动 Vite 的前端服务 + Electron 宿主
npm run dev
```

### 4. 项目打包 (Build)
如果你想把它变成一个独立的 `.exe` (Windows) 或 AppImage / snap (Linux)，运行：
```bash
npm run build
```

## 使用指南 📖

1. **唤醒控制台**：在桌面的小家伙身上 **右键单击**，即可打开拥有 5 大板块的全能控制台。
2. **连接大脑**：在【⚙️ 通用配置】中，填入你的 **DeepSeek API Key** (`sk-xxxxxx`)，保存后立即生效。
3. **互动与抚摸**：在它下方的输入框输入文字即可对话。如果你用鼠标按住它，还可以把它**拖拽**到屏幕的任意角落。
4. **管理记忆**：前往【🧠 记忆库管理】，你可以看到它偷偷记下的关于你的所有小秘密，你可以手动删减或者帮它补充。
5. **感知中心**：点击【👁️ 感知中心】，查看它当前监测到的桌面软硬件情况（Linux Wayland 如果权限不足会出现优雅的提示框）。

## 技术栈架构 🛠️

- **前端 UI**：React 18 + Vite，全局采用深色科幻风格 (Dark Mode Glassmorphism)，使用原生 CSS 构建丝滑组件。
- **桌面端宿主**：Electron，负责透明无边框窗口渲染、IPC 安全通信与持久化本地配置文件存储 (`pet-config.json`)。
- **大模型通信**：Axios 直连 DeepSeek 官方 API。
- **硬件探测接口**：`systeminformation` 与 `active-win`，用于捕获电量变化与活动窗口标题。

## 隐私与安全 🛡️

**你的数据，永远是你的。**
本软件所有的配置文件（包含你的提问记录、API Key、本地记忆信息）**完全不保存在项目源码文件夹中**，而是存在你的系统用户核心目录下（随系统而异，如 `~/.config/pet-chat/` 或 `AppData/Roaming/`）。
上传此代码库到 GitHub **绝对不会**发生隐私泄漏！
